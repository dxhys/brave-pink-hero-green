Kebijakan Privasi

Kami menaruh privasi Anda sebagai prioritas utama. Editor ini dibuat dengan prinsip privacy-first dan bekerja dengan cara berikut:

  • Tidak ada unggahan otomatis: Seluruh proses edit gambar dilakukan langsung di browser Anda menggunakan Canvas. Foto Anda tidak pernah meninggalkan perangkat.
  • Tidak ada penyimpanan: Kami tidak menyimpan atau mencatat gambar di server.
  • Proses server (opsional): Jika Anda memilih tombol “Proses di Server”, foto hanya diproses sementara di memori (RAM). Setelah selesai, hasil langsung dikirim kembali kepada Anda dan tidak pernah ditulis ke disk.
  • Tidak ada cache: Respon server menggunakan Cache-Control: no-store agar CDN/proxy tidak menyimpan salinan.
  • Transparansi: Anda bisa memverifikasi ini lewat kode sumber atau tab Network di browser.

Kami berkomitmen menjaga agar alat ini tetap aman, sederhana, dan ramah privasi.
